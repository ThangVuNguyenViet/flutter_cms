# Shadcn UI Flutter Components Documentation

## Overview
This directory contains comprehensive documentation for Shadcn UI Flutter components. These components provide a modern, customizable UI toolkit for Flutter applications.

## Available Components
- [Button](/components/button.md)
- [Input](/components/input.md)

## Quick Start
1. Install the Shadcn UI package:
```bash
flutter pub add shadcn_ui
```

2. Import the package in your Dart file:
```dart
import 'package:shadcn_ui/shadcn_ui.dart';
```

## Documentation Style
Each component markdown file includes:
- Overview

- Basic usage examples
- Advanced configuration options
- Best practices
- Performance considerations
